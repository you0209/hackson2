\section{担当D — 音響合成（Processing/Minim）}

\subsection{機能一覧}
本担当では，Arduino（担当C）から送信されるシリアル通信データを受信し，

Processing上で音響合成を行い，各楽器音をリアルタイムに出力する機能を実装する．

主な機能は以下の通りである．

\begin{itemize}

  \item シリアル通信受信機能  

  Arduinoから送信される3バイトパケット（FLAG / NOTE / VEL）を受信し，NOTE\_ONおよびNOTE\_OFFイベントとして解釈する．

  \item 受信データパース機能  

  受信したバイト列を3バイト単位で解析し，音名インデックスおよび音量情報を抽出する．

  \item 音高変換機能  

  音名インデックスをMIDIノート番号へ変換し，周波数に変換する．

  \item 音響合成機能  

  Minimライブラリを用い，Oscil・ADSR・フィルタ等を組み合わせたUGenチェーンにより音を生成する．

  \item 楽器音色生成機能  

  各楽器（ハイハット，バスドラム，鉄琴，ピアノ，トランペット）に応じた音色パラメータを設定する．

  \item 複合音生成機能  

  ハイハット＋シンバル，バスドラム＋スネアのように，1イベントで複数音を同時生成する．

  \item 発音・消音制御機能  

  NOTE\_ONで発音開始，NOTE\_OFFでADSRに従い減衰させる．

  \item スペクトラム解析による音色調整機能  

  外部ツールを用いて既存音源とのスペクトラム比較を行い，音色パラメータ（ADSR・フィルタ等）を調整する．

\end{itemize}
