# 課題4 — 人員ごとの作業時間統計

提出物4「人員ごとの作業時間の統計」用の章本体フォルダ．
親文書 `../reports.tex` の第1章として `\input` される．

## 構成

```
work_hours_member/
├── work_hours_member.tex   # 章本体（親文書から \input される）
└── README.md
```

## 記載内容

提出時点までの累計作業時間を，担当（学籍番号 : 氏名）ごとに記録する．
作業時間の集計は，担当者が個人作業ログから合算する．

## 追記の書式

`work_hours_member.tex` の `\endhead` と `\bottomrule` の間に，次の書式で行を追記する．

```
学籍番号 : 氏名 & XX 時間 & 主な作業内容 \\
```

例：

```
25G1099 : 長山 魁良 & 28 時間 & 楽譜データ構造の実装，シリアル通信プロトコル設計 \\
```

## ビルド方法

このフォルダ単独ではビルドしない．リポジトリルートで親文書をビルドする：

```bash
lualatex reports.tex
```

## 提出毎の更新

このファイルは **累計値** を記載するため，提出ごとに作業時間を更新する．
過去の提出値を残す必要がある場合は，列を追加するか別表に切り出して履歴を残すこと．
