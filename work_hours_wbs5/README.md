# 課題5 — WBS要素ごとの作業時間統計

提出物5「WBSの要素ごとの作業時間の統計」用の章本体フォルダ．
親文書 `../reports.tex` の第2章として `\input` される．

## 構成

```
work_hours_wbs/
├── work_hours_wbs.tex   # 章本体（親文書から \input される）
└── README.md
```

## 記載内容

WBS（作業分解構造）の各要素ごとに，提出時点までの累計作業時間を記録する．
WBS番号は `management_plan.tex` の WBS図と一致させること．

## 追記の書式

`work_hours_wbs.tex` の `\endhead` と `\bottomrule` の間に，次の書式で行を追記する．

```
WBS番号 & WBS要素名 & XX 時間 & 担当 \\
```

例：

```
A-1.2 & 指揮者人形ユニットのソフトウェア実装 & 12 時間 & 25G1xxx : 飛田 \\
```

## ビルド方法

このフォルダ単独ではビルドしない．リポジトリルートで親文書をビルドする：

```bash
lualatex reports.tex
```

## 課題4との関係

- 課題4（`work_hours_member/`）: 人員軸の集計
- 課題5（このフォルダ）: WBS軸の集計

両者は同じ作業時間の集計を「軸」を変えて切り出したものなので，
合計時間は基本的に一致するはず．不整合があれば集計ミスの可能性がある．
