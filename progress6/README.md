# 課題6 — 作業の進捗率

提出物6「作業の進捗率」用の章本体フォルダ．
親文書 `../reports.tex` の第3章として `\input` される．

## 構成

```
progress/
├── progress.tex   # 章本体（親文書から \input される）
└── README.md
```

## 記載内容

WBSタスクごとの進捗率（0〜100\%）を記録する．
タスク一覧は `wbs_v2-1780057657276.xlsx`（WBSシート）の構造に従う．

## 追記の書式

`progress.tex` の各タスク行を編集する．書式：

```
タスクID & タスク名 & 担当者 & 進捗率（%） & 備考 \\
```

例：

```
121 & サーボ機構設計 & 長見・飛田 & 80 & レビュー待ち \\
```

進捗率カラムを空のままにすると未着手扱いになる．数値（0〜100）を入れて更新する．

## ビルド方法

このフォルダ単独ではビルドしない．リポジトリルートで親文書をビルドする：

```bash
lualatex reports.tex
```

## 提出毎の更新

進捗率は提出のたびに最新の数値に書き換える．
備考欄に「テスト未実施」「レビュー待ち」などのステータスメモを残すと運用しやすい．
